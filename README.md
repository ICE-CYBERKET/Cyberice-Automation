# Cyberice-Automation
 Cybersecurity tools and experiments

**Cyberice-Automation** is a cybersecurity-focused automation project designed to simplify, speed up, and standardize common security tasks such as reconnaissance, scanning, analysis, and reporting.

This project is built with learning, experimentation, and real-world defensive/offensive security workflows in mind.
